git 指令介紹

先來看看在本地端建立 repo 的整個流程

本地端建立 repo 流程

  1. 新建一個新的 repo
  2. 新增/刪除檔案
  3. 將檔案加到追蹤對象
  4. commit 到 repo 中

Git 指令

  1. 在本地端新建 repo
  2. 使用指令新增/刪除檔案
    以 windows 終端機為例:
    以下指令均須 cd 到檔案或資料夾所在的上層位置
    新增資料夾 新增檔案 列出檔案下所有文件 刪除檔案 刪除特定副檔名的所有檔案 刪除空資料夾 刪除某資料夾下所有文件
  3. 將檔案加到追蹤對象 若要將全部加入可使用
  4. commit 到 repo 中

後面補充一些可以查看狀態的指令

  1. 查看文件狀態
    若出現圖片中指令代表需要加到追蹤對象
  2. 查看每次 commit，若要退出按下 q
